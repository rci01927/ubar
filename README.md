## uBar 是什么

简单说：它是一个 Dock 替代品，让你的 Mac 有一个长得像 Windows 任务栏的东西放在屏幕底部（或者顶部、左侧、右侧，随你）。

但 uBar 不只是换个皮。它比 Dock 能显示更多信息——每个正在运行的应用会展示名字，多窗口会分开显示，你不用猜"这个 Pages 图标下面打开了几个文件"。

## 核心功能有哪些

**任务栏 / Dock 两用**

uBar 可以同时配置成任务栏模式或 Dock 模式。显示标题 + 打开应用分组 = 任务栏；隐藏标题 + 全部分组 = Dock。看你心情，切来切去都行。

**窗口管理更清晰**

Dock 上的图标最大的毛病是不透明——你不知道这个应用开了几个窗口。uBar 直接把每个窗口单独列出来，或者把它们按应用分组，鼠标悬停还能预览窗口内容，省去了大量盲目切换的麻烦。

**楼层叠加，容纳更多**

任务栏一行放不下？没事，把鼠标移到 uBar 上边缘，三指向上拖，它会展开成多行——最多 5 层，密集恐惧症请绕行。

**收藏夹**

类似 Windows 任务栏的快捷方式功能，把常用的 App、文件夹钉在任务栏，随时点开，不用再满桌面找。

**「开始菜单」按钮**

左侧有一个类似 Windows 开始按钮的东西，点开能快速访问应用列表、文件夹，甚至直接重启、关机。第一次看到这个功能，有点恍惚，以为自己在用 Windows。

**多显示器支持**

每个屏幕都有自己的 uBar，只显示该屏幕上的应用和窗口。也可以开镜像模式，全屏幕同步显示相同内容。对多屏工作流来说很实用。

**主题自定义**

提供浅色、深色、Platinum Neue 等多套主题，也可以自定义颜色和透明度，想弄成什么风格大概都能实现。

**活动模式**

按住 Ctrl 键，uBar 上每个应用的 CPU 和内存用量直接显示出来，妥妥的实时任务管理器。

## 使用体验

uBar 是那种设计很细腻的软件。它没有强迫你完全抛弃 Dock，可以和 Dock 共存，让你慢慢适应。应用徽章（badge 数字）的显示比 Dock 清晰很多；不响应的应用会用红色背景标出来，不用猜"它是卡了还是在加载"。

软件本身很轻，没有明显的性能开销，在 Apple Silicon 机器上运行流畅。支持 macOS 10.13 以上，到最新的 macOS 15 Sequoia 都没问题。

当然，它也不是完美无缺。对 macOS 原生美学很在意的用户，可能需要一段时间接受任务栏的存在感比较强这件事。不过有 14 天试用，完全可以先摸清楚自己的喜好再决定。
